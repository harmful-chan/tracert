# tracert

## 说明
+ 为 Span 项目提供路由跟踪服务，可获取特定的公网IP发送并记录

## 使用方法
+ 路由跟踪
```
>tracert www.baidu.com
1 192.168.1.1
2 10.2.199.254
3 172.16.10.29
4 *
5 172.23.252.105
6 183.36.223.17
7 183.56.34.245
8 *
9 113.96.5.46
10 *
11 14.29.117.242
12 14.215.177.38
```
+ 获取特定地址
```
>tracert www.baidu.com 3
 172.16.10.29
```
